The Problem Statement

"A specialization platform for doctors (in specialized fields) to annotate images and get paid for annotation. The annotated data is used for training an ML model."

We built exactly this, but engineered for the trustless Web3 era.

Why Blockchain?

Traditional centralized data platforms require blind trust.

  • Escrow Contracts: Guarantee doctors are paid exactly what was promised, exactly when the threshold is met.
  • Soulbound Tokens (SBTs): Create immutable, non-transferable identities, tracking a doctor's lifelong accuracy and reputation score.
  • Regulatory Audit Trails: Every timestamp and annotation hash is stored on Polygon, perfect for FDA 510(k) submissions.

Decoupled Heterogeneous Stack

  • Async MLOps Core: Decouples heavy inference into Python backend via SSE Streams.
  • Zero-Latency UI: High-performance image loading using browser-level `IndexedDB` caching.
  • On-Chain Integrity: Immutable hashes and escrow-release logic secured on Polygon PoS.
  • Clinical Monolith: Industrial UI/UX designed for radiologists using modern React patterns.

Rapid Execution

What we built within the timeframe:

  • From zero to a full-stack MVP: 11+ complex, responsive frontend routes (Doctor portal, Company dashboard, Workspace, Admin panel).
  • Complete Web3 Integration: Authored, compiled, and deployed 3 independent Solidity Smart Contracts (Escrow, MockUSDC, DoctorSBT) and wired them to the frontend via Wagmi.
  • Fully Functional Python Backend: Built a standalone FastAPI server that coordinates real-time Epoch data across ports.
  • Working Pipeline: No smoke and mirrors. You can upload an image, annotate it, generate mathematically correct COCO JSON data, and mint an SBT on a live testnet.

Technical Depth & Complexity

  • Scalable Local Storage: Zero-latency image loading using `idb-keyval`, efficiently mapping massive DICOM/X-Ray blobs to Zustand state by reference to prevent browser OOM crashes.
  • Asynchronous MLOps Simulation: A FastAPI Python Backend streams live model epochs directly into our UI Terminal using Server-Sent Events (SSE) with dynamic custom CRLF stream parsers.
  • Regulatory Export Pipeline & DataViz: Built DOM-scraping algorithms (`html2canvas` + `jsPDF`) to compile interactive Recharts SVGs into A4 PDF Reports.
  • Decentralized P2P Escrow: Stripped out centralized admin keys, allowing data requesters independent, trustless release of USDC funds via blockchain consensus.

How It Works: The Startup

1. Upload Raw DICOM/Images
2. Local PHI Scrubbing (Edge)
3. Deposit USDC into Escrow
4. Real-time Training Monitoring
5. Export Validated Dataset

The Startup pipeline is designed for Total Regulatory Preparedness.

  • Uploads raw scans; our Python microservice strips Patient Info (PHI) locally.
  • Company defines target labels & funds the Polygon Smart Contract Escrow.
  • Monitors doctor progress & training epochs via live visual terminal.
  • Downloads expert-validated assets with a full Blockchain Audit Trail.

How It Works: The Doctor

1. Connect Wallet & Verify Credentials
2. Receive SBT Identity Token
3. Annotate via "Clinical Monolith" UI
4. Verify via Consensus Engine
5. Receive Instant USDC Payout

The Doctor experience focuses on Monetizing Medical Expertise fairly.

  • Doctors authenticate via Web3; credentials are verified by Governor.
  • Receives a Soulbound Token (SBT) that tracks lifespan accuracy.
  • Annotates images using AI-assisted tools (Smart Drafting & Clinical Captionist).
  • Payments are triggered instantly by smart contracts upon Consensus Approval.
